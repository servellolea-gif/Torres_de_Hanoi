# Torres_de_Hanoi
Código em Python que utiliza uma função recursiva para simular um jogo de Torres de Hanoi.
Torres de Hanoi é um jogo composto por três pinos fixos, onde podemos erguer torres de discos de raios diferentes. O objetivo do jogo é transportar a torre de discos de uma extremidade a outra utilizando o pino do meio como auxiliar. A regra principal é de que não se pode colocar um disco maior sobre outro.
